Django iTelegram
=================

It's a library for building Telegram bot on Django with extra built-in options.
created from `django-telegrambot`_ with a lot of extra features


Compatibility
=============

- Python: **3.6**, **3.7**, **3.8**
- Django: **3.0**, **3.1**, **3.2**
